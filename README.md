# Package Sorting application

## Running

`python app.py`


## Specifications

- Dimensions are given in `cm`  
- Weight is given is `kg`

## Classification

Bulky Package
- Volume (length x height x width) > 1,000,000
- length >= 150 or width >= 150 or height >= 150

Heavy Package
- mass >= 20

## Sorting

`REJECTED`: Bulky `and` Heavy

`SPECIAL`: Bulky `or` Heavy

`STANDARD`: Not Bulky `and` Not Heavy
